About this prototype

          
    
            
  • Single-file HTML/JS app — open index.html in any modern browser. No build step, no server.
    • 
            
  • All data (submissions, settings) stays in your browser's localStorage. Nothing is sent to GSS servers in this prototype.
    • 
            
  • Humanity actions (time clocks, new user, shifts) are mocked. Wiring them to the real Humanity API requires a backend with stored admin credentials.
    • 
            
  • AI Assistant calls Anthropic's API directly from your browser when a key is provided. For production, route through a backend so the key isn't exposed.
